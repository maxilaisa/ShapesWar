# Shape Arena Simulator

A physics-driven autonomous battle simulator where geometric shapes fight each other using AI decision-making.

## Features

- 12 unique shape fighters with fixed AI personalities
- Matter.js physics engine for realistic collisions
- PixiJS rendering with visual effects
- Combo system and ultimate abilities
- Real-time autonomous combat

## Tech Stack

- **Frontend:** React + TypeScript, Vite, Matter.js, PixiJS

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

## Deployment

Deploy to Render using the provided `render.yaml` configuration.
